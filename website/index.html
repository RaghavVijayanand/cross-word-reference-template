Continuous Digit Recognition

A project implementing Level Building algorithm for recognizing continuous digits using MFCC features and Dynamic Time Warping.

Learn Concepts View Results

Project Overview

This project demonstrates a complete pipeline for speech recognition of continuous digits (e.g., "1-2-3"). It moves from processing raw audio files to extracting features, building reference templates, and finally recognizing sequences using advanced dynamic programming techniques.

Key Features

  • MFCC Extraction: Converting audio waveforms into Mel-frequency cepstral coefficients.
  • Template Generation: Finding the "medoid" of training samples using DTW.
  • Level Building: A 2-level Dynamic Programming algorithm to recognize sequences of digits without segmentation.
  • Hybrid Implementation: Python for data processing and C for high-performance core algorithms.
