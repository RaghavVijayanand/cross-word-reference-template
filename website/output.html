Results & Analysis

Continuous Recognition Results (Team-1)

We tested the system with Single Template per Digit + Silence. Below is a sample of the output:

Filename        GroundTruth     Recognized
----------------------------------------
111a.mfcc       111             74
455a.mfcc       455             455
55a.mfcc        55              55
711a.mfcc       711             711
74a.mfcc        74              74
77a.mfcc        77              _77
...

Observation: We restricted the system to use Only One Template per Digit (the Medoid) plus a Silence template. The system correctly recognizes "455", "55", "711", "74", and "77". However, recognition of "111" degraded compared to the multi-template approach, suggesting that a single template for '1' struggles to capture the variance in connected speech.

Final Configuration:
1. Templates: Single Medoid per digit + Silence.
2. Constraints: Min 7 frames, Max 2x template length.
3. Penalties: 200.0.

Conclusion: The system works well for distinct digits (4, 5, 7) even with a single template, but '1' requires more variation or better template selection.

Visualization: Level Building Heatmap

The heatmap below visualizes the accumulated distance matrix D[l][t].

X-axis: Time frames of the test signal.
Y-axis: Level (number of digits recognized so far).
Color: Accumulated distance (Darker is lower cost/better match).

You can see the path traversing multiple levels, indicating that the algorithm prefers a sequence of digits over a single stretched digit.
